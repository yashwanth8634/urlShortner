# URL Shortener

URL shortener application for creating compact links from long URLs.

## Features

- Shorten long URLs
- Redirect short links to the original destination
- Lightweight and easy to use

## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

```bash
git clone <repository-url>
cd urlShortner
npm install
```

### Run

```bash
npm start
```

## Usage

1. Open the app in your browser.
2. Paste a long URL into the input field.
3. Generate a short URL.
4. Share or open the short link to redirect to the original URL.

## Project Structure

- `src/` - application source code
- `public/` - static assets
- `package.json` - project dependencies and scripts
